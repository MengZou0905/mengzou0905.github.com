博文加密

有一些博文不便公开，需要设置密码。

##hexo插件
hexo-blog-encrypt可以实现这个功能。github上的说明运行不成功，具体使用方法如下：

  • 安装
    npm install -g --save hexo-blog-encrypt
  • 设置根目录下的_config.yml
    加入encrypt: enable: true
  • 设置根目录下的package.json
    加入"hexo-blog-encrypt": "^2.0.10"
  • 修改需要被加密地博文
    开头处加入 password: abc abstract: 本文访问需要密码 message: 本文访问需要密码

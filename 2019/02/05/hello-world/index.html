我是如何搭建这个博客的

Github pages + hexo 是免费便捷的博客搭建方案。为什么选择这两者呢：Github pages为用户提供了静态网页展示系统，绑定地址https:username.github.io，如此一来，我们不必再掏钱去租用个人域名，就可以让博客在茫茫网络中有一个能被识别的位置；hexo是一套用于生成、发布静态网页的框架，用它来编排博客页面的内容、效果。

搭建环境为 Mac high sierra，中间踩坑若干（主要是node配置），附上亲测可跑通的博文，衷心感谢作者的分享，此处不再赘述。
